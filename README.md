# 🎮 Games Store — Next.js v.16

Proyecto de **tienda de videojuegos** desarrollado con **Next.js**, enfocado en aplicar buenas prácticas de desarrollo front‑end moderno y desplegado públicamente en Netlify.

## 🌐 Demo en producción

👉 **Ver la aplicación en vivo:**  
🔗 https://gaming-store-next.netlify.app/

## 📖 Descripción

**Games Store** es una aplicación web que simula una tienda online de videojuegos. El objetivo principal del proyecto es practicar y consolidar conocimientos en:

- Arquitectura con **Next.js (App Router)**
- Creación de componentes reutilizables
- Estilos modernos y diseño responsive
- Organización de un proyecto realista de front‑end
- Flujo completo desde desarrollo hasta despliegue

Este proyecto forma parte de mi crecimiento como desarrollador front‑end.
